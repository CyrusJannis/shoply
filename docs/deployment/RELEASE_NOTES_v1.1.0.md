# Release Notes v1.1.0

## 🎉 Neue Features

### Einkaufshistorie → Liste hinzufügen
- **Ganzen Einkauf zur Liste hinzufügen**: Klicke auf einen Einkauf in der Historie und füge alle Items mit einem Klick zur aktuellen Liste hinzu
- Perfekt zum Wiederholen von Einkäufen

### Verbesserte Sortierung
- **Custom Sort für Listen**: Sortiere deine Listen per Drag & Drop in der Reihenfolge, die du möchtest
- **Custom Sort für Items**: Sortiere Items innerhalb einer Liste per Drag & Drop
- Sortierung wird automatisch gespeichert

### Einkauf abschließen - Nur markierte Items
- Beim Abschließen eines Einkaufs werden **nur die markierten Items** in die Historie übernommen
- Nicht markierte Items bleiben in der Liste für den nächsten Einkauf

### Google Sign-In (Vorbereitung)
- Native Google Sign-In Integration vorbereitet
- Funktioniert in-app ohne Browser (nach Konfiguration)

## 🐛 Bug Fixes
- Verbesserte Performance bei großen Listen
- Stabilere Synchronisation mit Supabase

## 🔧 Technische Verbesserungen
- Firebase Messaging entfernt (Konflikte mit Google Sign-In)
- Aktualisierte Dependencies
- Code-Optimierungen
