# 🎯 DEIN PDF JETZT VERWENDEN - ANLEITUNG

## ✅ FERTIG EINGEBAUT! Scanner ist jetzt in deiner App!

### 📱 **Wo findest du den Scanner?**

1. Öffne deine Shoply App
2. Gehe zum **Profil-Tab** (ganz unten rechts)
3. Scrolle zu **"Data & Storage"**
4. Klicke auf **"🏷️ Prospekt Scanner"**

---

## 📂 **Wo legst du dein PDF hin?**

### **Option A: Direkt hochladen (EMPFOHLEN)**

**Du musst NICHTS in einen Ordner legen!**

1. Klicke im Scanner auf **"PDF Prospekt scannen"**
2. Der iOS/macOS File Picker öffnet sich automatisch
3. Navigiere zu deinem PDF:
   - Downloads
   - iCloud Drive
   - Dokumente
   - Überall wo dein PDF liegt!
4. PDF auswählen → Fertig!

Die App lädt es automatisch und scannt es.

## 🚀 **Schritt-für-Schritt: Dein erstes Prospekt scannen**

### **1. PDF besorgen**
- Gehe zu rewe.de/angebote
- Lade das aktuelle Prospekt als PDF herunter
- Speichere es in Downloads

### **2. App öffnen**
- Öffne Shoply
- Profil-Tab → "🏷️ Prospekt Scanner"

### **3. Supermarkt wählen**
- Wähle einen Chip aus: **REWE** ✅

### **4. PDF hochladen**
- Klicke "PDF Prospekt scannen"
- File Picker öffnet sich
- Wähle dein REWE PDF aus Downloads

### **5. Warten & Staunen**
```
📄 Konvertiere PDF zu Bildern...
🔍 Scanne Seite 3/8... (60%)
✅ 27 Angebote gefunden!
```

### **6. Angebote sehen**
Die App zeigt dir alle gefundenen Angebote:
```
🏷️ -30%  Weihenstephan Milch
REWE • Milchprodukte
1,99€ → 1,39€
Spare 0,60€
```

### **7. Automatisch gespeichert!**
- Alle Angebote sind jetzt in der Datenbank
- Wenn du "Milch" auf deine Einkaufsliste schreibst...
- ...zeigt die App automatisch: **🏷️ -30%** neben dem Produkt!

## 💡 **Tipps für beste Ergebnisse**

✅ **DO:**
- Offizielle Prospekt-PDFs verwenden
- Klare, hochauflösende PDFs (mindestens 300 DPI)
- PDFs mit gut lesbaren Preisen
- Deutsche Prospekte

❌ **DON'T:**
- Keine Fotos von Prospekten (nutze richtige PDFs)
- Keine Screenshots
- Keine zu niedrige Auflösung

---

## 🐛 **Probleme?**

### **"Keine Angebote gefunden"**
- Prüfe: Sind Preise im Format "2,99 €" vorhanden?
- Scanne ein anderes PDF zum Testen
- Versuche REWE oder Lidl PDFs (funktionieren am besten)

### **"Sprachdaten werden heruntergeladen"**
- Beim **ersten Start** lädt die App ~10MB Sprachdaten
- Das dauert ~30 Sekunden
- Danach nie wieder nötig!

### **"File Picker öffnet sich nicht"**
- Prüfe: Hast du einen Supermarkt ausgewählt? (Chip muss blau sein)
